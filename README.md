# 픽셀 아트 에디터 (PIXEL MAKER)

## 문제 · 사용자

기존 픽셀 아트 제작 도구는 설치가 필요하거나 복잡한 UI 때문에 진입 장벽이 높았습니다.
이 앱은 브라우저에서 바로 픽셀 아트를 만들고, 크기 설정부터 저장·불러오기·내보내기까지 한 번에 할 수 있는 웹앱입니다.

대상 사용자: 설치 없이 브라우저에서 바로 픽셀 아트를 만들고 싶은 사용자, 원하는 용지 비율과 크기를 자유롭게 정하고 싶은 사용자, 만든 작품을 파일로 저장하거나 이어서 작업하고 싶은 사용자.

## 핵심 기능 3개

1. **픽셀 단위 직접 드로잉**
   - 펜 / 지우개 / 스포이드 도구로 픽셀을 직접 편집
   - 브러시·지우개 크기를 팝업으로 조절 (크기 프리뷰 제공)
   - 마우스 휠로 캔버스 확대·축소, Undo/Redo 지원

2. **자유로운 용지 설정**
   - A/B 규격, 16:9, 1:1, 32:1, 사용자 지정 등 다양한 비율 프리셋
   - 가로·세로 픽셀 수 직접 입력 가능
   - 초기 화면에서 비율 미리보기 제공

3. **내보내기 · 불러오기**
   - PNG / JSON 형식으로 저장
   - 저장한 파일을 다시 불러와 이어서 작업 가능
   - JSON 또는 PNG 파일을 통합 불러오기 팝업으로 선택
