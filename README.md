# MediaConverter

무료 온라인 미디어 컨버터 웹앱입니다. 미디어 파일을 설치 없이 웹에서 업로드하고, 원하는 형식으로 변환한 뒤 다운로드할 수 있도록 기획한 프로젝트입니다.

## 미디어 컨버터

브라우저에서 바로 사용할 수 있는 온라인 미디어 변환 웹앱입니다. 서로 다른 파일 형식 때문에 재생·업로드가 되지 않는 문제를 줄이고, 별도 소프트웨어 설치 없이 빠르게 형식을 바꾸는 것을 목표로 했습니다.

## 문제 · 사용자

### 해결하려는 문제
- 미디어 파일 형식이 달라 웹·모바일에서 재생 또는 업로드가 되지 않는 경우가 있음.
- 기존 앱은 사용량 제한이나 광고 등으로 사용성이 떨어질 수 있음.
- 간단한 변환 작업에도 무거운 소프트웨어를 설치해야 하는 번거로움이 있음.
- 다양한 미디어 컨버터가 흩어져 있어 한 곳에서 통합적으로 사용하기 어려움.

### 주요 사용자
- 대학생 / 직장인: 과제 및 업무용 파일 형식 변환이 필요한 사용자
- SNS 콘텐츠 제작자: 플랫폼별 호환 형식 변환이 필요한 사용자
- 일반 사용자: 설치 없이 웹에서 빠르게 변환하고 저장 공간도 줄이고 싶은 사용자

## 핵심 기능 3개

### 1. 파일 업로드 및 변환
- 미디어 파일 업로드
- 대상 형식 선택
- 변환 시작
- 예: MKV → MP4, WAV → MP3, PNG → JPG

### 2. 변환 파일 다운로드
- 변환 완료 후 다운로드 버튼 제공
- 결과 파일을 바로 저장 가능

### 3. 사용자 인터페이스
- 드래그 앤 드롭 파일 업로드
- 형식 선택 및 옵션 설정
- 변환 진행률 프로그레스바
- 에러 메시지 명확히 표시

## 입력 / 출력 예시

### 입력 데이터 예시
- 파일명: lecture.mkv
- 형식 / 크기: MKV, 245MB
- 해상도: 1920×1080, 30fps
- 오디오: AAC
- 변환 형식: MP4
- 비디오 비트레이트: 5Mbps
- 오디오 비트레이트: 192kbps

### 출력 데이터 예시
- 파일명: lecture_converted.mp4
- 형식 / 크기: MP4, 198MB
- 해상도: 1920×1080, 30fps
- 오디오: AAC
- 크기 감소: 19.2%
- 변환 시간: 42초
- 결과 화면: 완료 메시지 + 다운로드
