# 픽셀 아트 에디터

## 문제
기존 픽셀 아트 제작은 정해진 캔버스나 복잡한 도구에 의존해서, 직접 픽셀 단위로 편집하고 저장하는 흐름이 불편했습니다.  
이 프로젝트는 브라우저에서 바로 픽셀 아트를 만들고, 크기 설정부터 저장·불러오기·내보내기까지 한 번에 할 수 있게 만든 웹앱입니다.  
웹 기반 픽셀 에디터는 그리드 기반 드로잉, 색상 선택, 이미지 불러오기·내보내기 기능을 제공하는 방식이 일반적입니다 [web:181][web:5].

## 사용자
- 픽셀 아트를 직접 만들고 싶은 사용자.
- 원하는 용지 비율과 크기를 자유롭게 정하고 싶은 사용자.
- 브라우저에서 간단하게 실행되는 편집기를 원하는 사용자.
- 만든 작품을 파일로 저장하거나 다시 불러와 이어서 작업하고 싶은 사용자.

## 핵심 기능 3개
1. **픽셀 단위 직접 드로잉**
   - 펜, 지우개, 스포이드로 픽셀을 직접 편집할 수 있습니다.
   - 브러시 크기를 정사각형 단위로 조절할 수 있습니다.

2. **자유로운 용지 설정**
   - A/B 규격, 16:1, 32:1, 사용자 지정 등 여러 비율을 고를 수 있습니다.
   - 가로·세로 픽셀 수를 직접 지정할 수 있습니다.

3. **내보내기 / 불러오기**
   - PNG, JSON 형식으로 저장할 수 있습니다.
   - 저장한 파일을 다시 불러와 이어서 작업할 수 있습니다.
