# 개선 기록 - 픽셀 아트 에디터 웹앱

## 개선 기록 1
**기본 기능:**
1. 픽셀 아트를 직접 찍어서 제작할 수 있다
2. 픽셀 아트의 용지 크기를 자유롭게 설정할 수 있다
3. 화면 상단 퀵바에 펜, 지우개, 색 선택, undo&redo 기능 있으며, 펜과 지우개는 크기 조정 가능 (1×1, 2×2 정사각형 단위)
4. 제작한 픽셀아트를 다양한 형식으로 내보내기 가능

**앱 흐름:**
주소 접속 → 세팅창 (용지 크기, 비율 설정) → 시작 버튼 → 제작 창 → 내보내기 → 다운받기

**초기 화면 인터페이스:**
- 상단: 'PIXEL MAKER' 제목
- 비율 선택 드롭다운: A 규격, B 규격, 16:1, 32:1, 사용자 지정 (마지막)
- 드롭다운 밑: 가로×세로 비율 조정 (자동 수정)
- 드롭다운 옆: 비율 예시 화면 (흰색 사각형)
- 하단: '시작' 버튼

**제작 창 인터페이스:**
- 픽셀 가이드라인 새겨진 용지 (옅은 회색)
- 스크롤: 화면 크기 조정, 왼클릭: 색칠, 우클릭: 지우개, 가운데 클릭: 화면 이동
- 상단 태스크바: 펜, 지우개, 색, 불러오기, 내보내기 버튼
- 펜/지우개: 팝업창으로 사이즈 조정
- 색: RGB 정사각형, 위쪽 무지개 기본색, 오른쪽하단 스포일러 마크
- 색 조정: RGB, 색상코드, 채도, 명도, 감마
- 내보내기: 팝업으로 파일 형식 리스트 선택
- 불러오기: 특정 형식 파일 이어서 작업

**테마:**
- 라이트 테마: 베이지 색
- 다크 테마: 어두운 남색

**배포:**
GitHub 레포지토리에 업로드, 주소 접속 시 실행

## 개선 기록 2
**팝업 기능 개선:**
- 색 선택창: 색을 누르면 팝업으로 떠야 함 (평소 아님)
- 펜 크기: 펜 선택 후 한 번 더 클릭 → 팝업으로 조절
- 지우개 크기: 지우개 선택 후 한 번 더 클릭 → 팝업으로 조절
- 모두 지우기: 지우개 선택 후 한 번 더 누름 → 팝업으로 떠음
- 내보내기: 버튼 누르면 팝업으로 리스트 뜨고 선택 (툴바에 따로 노출 안 됨)

**툴바 통합:**
- 불러오기: PNG 불러오기와 통합 (單獨 분리 안 됨)

**다크/라이트 토글:**
- 모든 화면 오른쪽 상단 구석에 해/달 모양 토글 (초기화면만 아님)

**새로운 버튼:**
- 설정 버튼: 툴바 맨 오른쪽, 톱니바퀴 모양

**툴바 디자인:**
- 북마크 바처럼 영역 구분
- 라이트: 배경 흰색, 다크: 배경 검은색

**기타:**
- 용지: 항상 화면 가운데 배치
- 펜/지우개 크기: 팝업에서 프리뷰 제공
- 초기화면 프리뷰: 내부색 흰색으로 꽉 채움

## 개선 기록 3
**새로운 버튼:**
- 손모양 버튼 추가: 선택 후 그랩으로 움직이는 기능

**비율 드롭다운 변경:**
- A 규격, B 규격 → ISO(1:√2) 로 통합 (√2 는 수학기호)
- 32:1 제거
- 추가: 16:9, 21:9, 32:9, 4:3, 5:4
- 가로/세로 전환 버튼 추가 (비율 전환 가능)

**툴바 디자인 개선:**
- 설정 버튼: 라이트 토글과 겹침 → 좌측에 토글 버튼 배치
- 버튼 이모지: 어색함 → 다른 적절한 대체제 또는 다른 디자인 방향성 적용

**불러오기 기능 확장:**
- 사진 → 픽셀 변환 불러오기
- 배경 종이를 사진으로 바꾸는 불러오기 추가

## 개선 기록 4
**버튼 디자인 세부:**
- 설정 기호: 해모양 같음 → 더 톱니바퀴처럼 변경
- 이동 기호: 십자모양 4 끝에 전부 화살표 달린 모양

**내보내기 기능:**
- JPG 내보내기 추가
- PNG: 배경 없음 (투명), JPG: 배경과 함께
- JPG 옵션: 격자선 포함/미포함 선택

**입력란 개선:**
- 변환버튼 크기: 부자연스러움 → 가로/세로 픽셀 수 입력란과 같은 높이를 변으로 가진 둥근 사각형으로 변경 (위치 동일)
- 픽셀 입력: 숫자 지워도 0 사라지지 않음 → 입력 시 0 지워지고, 공란/숫자 아님 시 0 표기

**테마:**
- 다크 모드에서도 용지 배경색: 흰색 유지

## 개선 기록 5
**색 선택 시스템 통합:**
- 현재: 한 번 클릭 → 기본색, 두 번 클릭 → 상세 조절 (불편)
- 개선: 툴바에서 색 누르면 하나의 통합 팝업창으로 모든 기능 제공

**통합 팝업 구조:**
1. 가장 상단: 기본색 한 줄
2. 그 아래: 색상 사용 히스토리 한 줄
3. 그 아래: 그라디언트 필드로 색상 선택 공간
4. 가장 아래: 그라디언트 필드 연동된 [색상, 명도, 채도] 조절 가로 슬라이더 바 3 줄

## 전체 요약

**총 기록 수:** 6 개

**주요 개선 카테고리:**
1. 팝업 기능: 색 선택, 펜/지우개 크기, 모두 지우기, 내보내기
2. 툴바 구성: 손모양 버튼 추가, 설정/토글 배치 변경, 버튼 통합
3. 비율 시스템: ISO 통합, 새 비율 추가, 전환 버튼
4. 디자인: 버튼 이모지 개선, 툴바 영역 구분, 용지 중앙 배치
5. 색 선택: 통합 팝업 시스템 (기본색, 히스토리, 그라디언트, HSV 슬라이더)
6. 내보내기: PNG 투명 배경, JPG 격자선 옵션
7. 입력란: 변환버튼 모양, 픽셀 입력 0 처리

**최종 목표:** GitHub 에 업로드하여 웹 접속 실행 가능한 픽셀 아트 에디터 웹앱
