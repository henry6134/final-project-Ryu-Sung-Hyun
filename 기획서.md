# MediaConverter 기획서

## 1. 프로젝트 개요

**MediaConverter**는 무료 온라인 미디어 컨버터 웹앱입니다. 사용자가 브라우저에서 바로 미디어 파일을 업로드하고, 원하는 형식으로 변환한 뒤 다운로드할 수 있도록 기획한 프로젝트입니다.

이 프로젝트의 목적은 파일 형식 호환성 문제를 줄이고, 복잡한 설치 과정 없이 간단하고 빠른 미디어 변환 경험을 제공하는 것입니다.

## 2. 해결하려는 문제

현재 사용자는 다양한 미디어 형식 차이로 인해 여러 불편을 겪고 있습니다.

- 미디어 파일 형식이 달라 웹·모바일 환경에서 재생 및 업로드가 되지 않는 경우가 많음
- 기존 변환 서비스는 광고, 사용량 제한 등으로 불편함이 있음
- 단순 변환 작업에도 무거운 프로그램을 별도로 설치해야 함
- 미디어 변환 서비스가 여러 곳에 흩어져 있어 접근성이 떨어짐

## 3. 주요 사용자

### 대학생 / 직장인
- 과제 제출용 파일 형식 변환
- 업무 문서 및 미디어 파일 형식 변경

### SNS 콘텐츠 제작자
- 플랫폼 업로드용 호환 형식 변환
- 용량 최적화 및 형식 통일 필요

### 일반 사용자
- 설치 없이 웹에서 빠른 변환 희망
- 미디어 파일 용량을 줄여 저장 공간 절약

## 4. 핵심 기능 3개

### 4-1. 파일 업로드 및 변환
사용자는 파일을 업로드한 후 변환 대상 형식을 선택하고 변환을 시작할 수 있습니다.

예시:
- MKV → MP4
- WAV → MP3
- PNG → JPG

### 4-2. 변환 파일 다운로드
변환이 완료되면 결과 파일을 다운로드할 수 있는 버튼을 제공합니다.

### 4-3. 사용자 인터페이스
사용 편의성을 높이기 위해 다음 요소를 제공합니다.

- 드래그 앤 드롭 업로드
- 형식 선택 및 옵션 설정
- 변환 진행률 프로그레스바
- 명확한 에러 메시지 표시

## 5. 입력 / 출력 데이터 예시

### 입력 데이터 예시
- 파일명: lecture.mkv
- 형식 / 크기: MKV, 245MB
- 해상도: 1920×1080, 30fps
- 오디오: AAC
- 변환 형식: MP4
- 비디오 비트레이트: 5Mbps
- 오디오 비트레이트: 192kbps

### 출력 데이터 예시
- 파일명: lecture_converted.mp4
- 형식 / 크기: MP4, 198MB
- 해상도: 1920×1080, 30fps
- 오디오: AAC
- 크기 감소: 19.2%
- 변환 시간: 42초
- 결과 화면: 완료 메시지 + 다운로드

## 6. 테스트 시나리오

### 시나리오 1 - 성공
1. `song.wav` (WAV, 45MB, 44.1kHz) 업로드
2. 대상 형식 `MP3`, 비트레이트 `192kbps` 선택
3. `변환 시작` 클릭

**예상 결과**
- `song_converted.mp3` 생성 (4.2MB)
- 크기 감소 90.7%
- 변환 시간 18초
- 다운로드 버튼 활성화
- 히스토리 1건 기록

### 시나리오 2 - 실패
1. `video_large.mov` (MOV, 1.8GB) 업로드
2. 대상 형식 `MP4` 선택
3. `변환 시작` 클릭

**예상 결과**
- 파일 크기 제한 초과 안내
- `파일 크기가 1GB를 초과합니다. 현재: 1.8GB` 메시지 출력
- 변환 시도 없음
- 히스토리 0건 유지

## 7. 현재 구현 상태

### 프론트엔드
- React 기반 기본 UI 구현

### 백엔드 / 변환 엔진
- Node.js + FFmpeg 기반 로컬 변환 테스트 진행

### 배포 상태
- Vite 기반 로컬 웹 서버 실행 가능
- 로컬 링크: http://localhost:3000

## 8. 실행 방법

```bash
cd {저장경로}
npm install
npm run build
npx serve dist
```

※ Node.js 설치 필요

## 9. 기대 효과

- 별도 프로그램 설치 없이 브라우저에서 파일 변환 가능
- 파일 형식 호환성 문제 해결
- 다양한 사용자층이 빠르게 접근 가능
- 결과 다운로드와 에러 처리까지 한 화면에서 제공 가능
